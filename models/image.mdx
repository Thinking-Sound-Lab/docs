Invook supports a range of powerful image generation models. Each model uses a specific number of **credits** per generation.

<Note>
  Credits are deducted from your account balance each time a model runs. Free plan users receive **300 credits/month**. See [FAQs](/help/faqs) for more details.
</Note>

---

## Available Image Models

| Model Name | Credits per Generation |
|---|---|
| **Imagen 4** | 9 credits |
| **Nano Banana Pro** | 37 credits |
| **Ideogram v3 (Turbo)** | 7 credits |
| **Recraft V4 (Normal)** | 9 credits |
| **Flux Kontext** | 9 credits |
| **Bria Fibo Relight** | 9 credits |
| **Topaz Upscale** | 18 credits |

---

## How to Use Image Models

1. Add an **Image Model** node to your Canvas using the [Add Node](/support/toolbox#3-add-node) tool
2. Connect a **File** node (your input image) and a **Prompt** node
3. Run the model to generate a new image

<Tip>
  New to the Canvas? Follow the [Create Your First Asset](/tutorial/create-first-asset) guide for a full walkthrough.
</Tip>
